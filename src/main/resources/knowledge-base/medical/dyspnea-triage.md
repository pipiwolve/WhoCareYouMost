# 呼吸困难危险信号

呼吸困难可由心肺、血栓、感染、过敏等多种原因引起，不能仅凭主观“气短”自行判断轻重，应优先识别危重征象。

## 立即急诊

如果出现严重呼吸困难（说话困难或明显喘不过气）、胸部压榨痛、口唇或皮肤发青发灰、意识混乱、晕厥，应立即呼叫急救。

## 需要尽快就医的信号

若伴咯血、单侧下肢肿痛、心跳明显不齐或过快、平卧即明显憋气，应尽快线下评估，警惕肺栓塞、心衰或严重心律失常。

## 居家观察边界

仅在症状轻微、短时可缓解且无危险信号时可短时观察；若夜间加重、活动耐量持续下降或反复发作，应尽快就医。
