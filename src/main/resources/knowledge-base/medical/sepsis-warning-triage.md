# 败血症危险信号

败血症是感染引发的危及生命状态，可在短时间内进展为器官功能衰竭。出现感染背景下的全身恶化表现时要高度警惕。

## 立即急诊

若出现意识混乱、呼吸急促、皮肤湿冷或花斑、心率明显增快、持续高热或寒战、极度虚弱，尤其合并低血压或少尿，应立即急诊。

## 高风险人群

老年人、婴幼儿、孕产妇、肿瘤治疗中、免疫功能低下及合并慢病者，一旦感染后状态快速变差，不建议居家观察。

## 就医原则

败血症强调尽早识别与尽快治疗。线上咨询仅可用于初筛，不应替代线下紧急评估。
