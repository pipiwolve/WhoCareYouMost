# 月经异常评估要点

月经相关不适需关注出血量、持续时间、伴随腹痛程度和是否出现头晕乏力。周期突然改变或经量明显增加需要进一步评估。

## 需要立即就医的信号

出现短时间大量阴道流血、明显头晕乏力接近晕厥、持续剧烈下腹痛、发热或恶臭分泌物时，应尽快急诊。

## 需要尽快妇科就诊的情形

连续多个周期紊乱、经期延长、经间期反复出血、痛经进行性加重影响日常生活时，应尽快妇科评估。

## 居家处理

无危险信号时可记录月经日历和症状变化、规律作息并避免过度劳累；症状持续恶化应及时就医。
