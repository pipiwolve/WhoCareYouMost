# 过敏反应危险信号

荨麻疹可表现为风团和瘙痒，但若合并呼吸道、循环系统或消化系统急性症状，需警惕过敏性休克。

## 立即急诊

出现喉头紧缩感、喘鸣、呼吸困难、口唇舌体肿胀、头晕濒晕、血压下降或短时间内多系统症状，应立即急诊。

## 不建议仅居家观察的情形

首次严重过敏反应、既往有过敏性休克史、儿童或老年人快速进展性症状时，不建议自行观察等待。

## 居家处理

轻度局限性皮疹且无系统症状时可暂时脱离可疑诱因并观察，但若症状扩展或再次发作应尽快就医。
