# 心理危机危险信号

若出现明确轻生想法、计划或行为，或情绪在短期内急剧恶化伴绝望、冲动和社会退缩，应按紧急风险处理。

## 立即急诊

发生自伤自杀行为、存在即刻自杀计划且具备实施条件、出现严重意识障碍或合并药物过量/胸闷气促等躯体急症时，应立即急救。

## 需要紧急专业支持的情形

出现“活着没有意义”、反复谈及死亡、行为明显异常、酒精药物滥用加重或家属无法保证安全时，应尽快精神专科评估。

## 安全原则

危机期应尽量避免独处，移除高风险工具，尽快联系当地心理危机干预热线或急诊系统。
