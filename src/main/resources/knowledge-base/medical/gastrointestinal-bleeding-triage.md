# 消化道出血危险信号

消化道出血可表现为呕血、咖啡色呕吐物、黑便或鲜血便，严重时可出现休克表现，应按急症处置。

## 立即急诊

出现呕血、黑便伴头晕乏力、心悸、面色苍白、出冷汗、意识模糊，或大量便血时，应立即急诊。

## 不建议仅居家观察的情形

老年人、正在使用抗凝抗血小板药物、肝病或既往消化道溃疡患者出现疑似出血时，不建议仅自行观察。

## 就医前注意

等待救治期间避免进食饮水，保留可疑呕吐物或粪便特征信息，便于医生快速判断出血部位和严重程度。
