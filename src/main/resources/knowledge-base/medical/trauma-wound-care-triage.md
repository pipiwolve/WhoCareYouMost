# 外伤评估要点

外伤分诊应先判断生命体征稳定性，再评估出血量、伤口深度、污染程度以及是否合并骨折、神经血管损伤或头部外伤。

## 立即急诊

出现活动性大出血、伤口深且污染重、疑似骨折畸形、伤后意识改变、反复呕吐、剧烈头痛或胸腹部疼痛时，应立即急诊。

## 需要尽快处理的情形

动物或人咬伤、伤口超过一定时间未处理、破伤风免疫状态不清、糖尿病或免疫抑制患者受伤时，应尽快线下处理。

## 居家处理

轻微表浅伤口可先清洁压迫止血并覆盖清洁敷料，但若红肿热痛加重、渗液或发热，应及时就医。
