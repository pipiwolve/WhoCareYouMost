# 发热与咳嗽的居家观察

成人发热伴咳嗽时，应关注体温变化、精神状态、补液情况以及是否出现呼吸困难、胸痛、意识改变等危险信号。

## 需要尽快就医的信号

如果体温持续在三十九摄氏度以上且反复不退，或者出现气喘加重、胸闷、呼吸频率明显增快、口唇发紫、精神反应差，应尽快就医。

## 居家处理

在没有危险信号时，可以先补液、休息、监测体温，必要时在确认禁忌和过敏史后使用常规退热药。但如果症状进行性加重，应及时线下评估。
