# 血栓相关危险信号

深静脉血栓可表现为单侧肢体肿胀、疼痛、压痛、皮温升高或皮肤颜色改变；血栓脱落可导致致命性肺栓塞。

## 立即急诊

出现突发呼吸困难、吸气或咳嗽时胸痛加重、咯血、心跳明显过快或不齐、头晕晕厥时，应立即急诊。

## 需要尽快就医的信号

出现不明原因单侧小腿或上肢肿痛、发热发红且非外伤所致时，应尽快就医排查静脉血栓。

## 风险提示

术后、长期卧床、肿瘤、妊娠产后、既往血栓史人群若出现上述症状，需更早升级线下评估。
