# 皮疹评估要点

皮疹评估应关注起病速度、分布范围、是否伴发热和黏膜受累，以及近期是否有新药、食物、感染或接触史。

## 立即急诊

出现全身快速扩散皮疹伴呼吸困难、口唇舌体肿胀、高热、眼口生殖器黏膜糜烂或皮肤大片脱落时，应立即急诊。

## 需要尽快皮肤科就诊的情形

皮疹持续超过一周、反复发作、夜间瘙痒明显影响睡眠、合并渗液结痂或继发感染时，应尽快就诊。

## 居家处理

无危险信号时应避免抓挠、减少可疑过敏原接触、保持皮肤保湿和清洁；若加重需线下评估。
