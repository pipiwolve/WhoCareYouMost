# 脑卒中预警信号

出现口角歪斜、言语不清、单侧肢体无力或麻木、突发视物模糊、行走不稳时，应高度怀疑脑卒中并立即启动急救流程。

## 立即急诊

一旦出现上述任何一个急性神经功能缺损症状，无论是否短暂缓解，都应立即呼叫急救并前往有卒中救治能力的医院。

## 不建议等待观察的原因

卒中治疗窗口期明确，延误就医会显著增加残疾和死亡风险。短暂性脑缺血发作也可能在短期内进展为完全卒中。

## 现场处理

等待急救时应保持平卧或侧卧、避免进食饮水、记录症状起始时间，不要自行驾车前往医院。
