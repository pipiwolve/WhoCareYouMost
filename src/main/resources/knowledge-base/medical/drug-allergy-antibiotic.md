# 药物过敏史的重要性

在考虑使用任何抗菌药物前，应先确认既往过敏史，尤其是是否对青霉素、头孢类或其他常见抗菌药出现过皮疹、呼吸困难或过敏性休克。

## 对青霉素过敏时

已明确对青霉素过敏的患者，不应自行随意使用来源不明的抗菌药，也不应忽视交叉过敏风险，应由医生结合病情、既往反应严重程度和替代方案综合判断。

## 何时需要紧急处理

如果服药后出现全身荨麻疹、口唇舌体肿胀、喘鸣、呼吸困难、头晕或血压下降，应立即停药并紧急就医。
