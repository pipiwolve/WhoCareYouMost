# 鼻出血初步处理

多数鼻出血可通过坐位前倾并持续捏压鼻翼十到十五分钟缓解，但需识别持续出血和全身不稳等高风险情形。

## 立即急诊

若鼻出血持续超过十到十五分钟仍不止、出血量大、伴头部外伤、头晕乏力、呼吸困难或反复呕血感，应立即急诊。

## 需要尽快就医的情形

反复鼻出血、正在使用抗凝药、合并出血倾向疾病者，应尽快耳鼻喉科评估。

## 居家注意事项

出血停止后二十四小时内避免抠鼻、用力擤鼻、剧烈运动和热刺激，减少再次出血风险。
