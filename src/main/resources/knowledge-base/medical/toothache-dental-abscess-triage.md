# 牙痛风险评估

牙痛常见于龋齿、牙髓或根尖感染。疼痛持续、面颊肿胀或发热提示感染进展风险，不应长期拖延。

## 需要尽快就诊的情形

牙痛持续超过两天、止痛后反复加重、伴咬合痛、口腔异味或颊部肿胀，应尽快口腔科就诊。

## 立即急诊

若出现眼眶周围或颈部肿胀，或口腔颈部肿胀导致呼吸、吞咽、说话困难，应立即急诊处理。

## 居家处理边界

短期可采用基础镇痛和口腔清洁措施缓解，但不应以反复自行用药替代口腔感染病灶处理。
