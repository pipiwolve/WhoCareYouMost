# 胸闷胸紧分诊重点

胸闷可由惊恐发作引起，但在未排除心肺急症前，不应先入为主判断为心理问题。

## 立即急诊

若胸闷伴胸痛放射、呼吸困难加重、出冷汗、晕厥、神经功能异常或持续不缓解，应立即急诊。

## 需要尽快评估的情形

反复发作影响功能、伴心悸或夜间憋醒、既往有心肺基础病者，应尽快线下完善评估。

## 安全处理原则

可在保证安全前提下进行呼吸放松，但任何“与既往惊恐体验不同”的发作均应优先排除躯体急症。
