# 便秘评估要点

便秘需关注排便频率变化、粪便性状、排便困难程度及是否伴腹痛腹胀、便血、体重下降或贫血表现。

## 需要尽快就医的信号

新近发生且持续加重的便秘、便血或黑便、进行性体重下降、持续腹痛呕吐、老年人排便习惯突然改变时，应尽快就医。

## 立即急诊

若出现严重腹胀伴停止排气排便、持续呕吐、剧烈腹痛或意识差，需急诊排除肠梗阻等紧急情况。

## 居家处理

无危险信号时可增加饮水和膳食纤维、规律活动和排便训练；若反复发作应门诊明确病因并规范治疗。
