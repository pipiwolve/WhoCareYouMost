# 腹痛危险信号

腹痛评估要关注疼痛部位、持续时间、是否进行性加重，以及是否伴随发热、呕吐、便血、黑便、黄疸或腹部压痛反跳痛。

## 立即急诊

如果出现突发剧烈腹痛、腹部僵硬、持续呕吐不能进水、呕血或黑便、便血、意识改变、血压下降或晕厥，应立即前往急诊。

## 不建议仅居家观察的情形

老年人、孕妇、儿童、肿瘤或免疫抑制人群出现持续腹痛时，即使症状不典型，也不建议长期居家观察，应尽快线下评估。

## 居家处理

在无危险信号且症状轻微时，可短期清淡饮食、补液、休息并观察二十四小时；若疼痛范围扩大或反复发作，应及时就医。
