# 焦虑惊恐症状识别

惊恐发作可出现心慌、胸闷、呼吸急促、濒死感和手足麻木。分诊时需先排除心肺和神经系统急症，再考虑心理因素。

## 立即急诊

若症状伴持续胸痛、明显呼吸困难、晕厥、神经功能缺损、意识异常或自伤风险，应立即急诊处理。

## 需要尽快门诊评估的情形

反复惊恐发作影响工作睡眠、持续焦虑超过数周、伴明显回避行为或抑郁表现时，应尽快精神心理专科评估。

## 自我调节与支持

无急症信号时可通过规律作息、减少咖啡因、呼吸放松训练和社会支持减轻症状；症状迁延需专业干预。
