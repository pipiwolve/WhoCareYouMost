# 腹胀消化不良评估

腹胀和消化不良常与饮食、胃肠动力或功能性疾病相关，但需筛查进行性体重下降、贫血、呕血黑便等器质性风险信号。

## 需要尽快就医的信号

出现持续上腹痛、反复呕吐、吞咽困难、呕血或黑便、短期体重明显下降、夜间痛醒时，应尽快就医。

## 不建议长期自行用药

症状超过两周反复发作时，不建议长期自行使用抑酸或促动力药，应由医生评估诱因和并发症风险。

## 居家处理

无危险信号时可少量多餐、减少油腻辛辣和酒精、避免睡前进食并记录诱发食物；症状迁延需门诊评估。
