# 哮吼危险信号

哮吼常见犬吠样咳嗽、声音嘶哑和吸气性喉鸣，夜间可加重。多数轻症可自限，但需警惕呼吸道梗阻风险。

## 立即急诊

若出现明显呼吸费力、胸壁凹陷、口唇皮肤发青发灰、嗜睡难唤醒、流涎吞咽困难或反应异常，应立即急诊。

## 需要尽快就医的情形

疑似哮吼且症状逐步加重、经初步处理后无改善、年龄过小或家长难以判断病情变化时，应尽快线下评估。

## 居家观察边界

轻症时以安抚、补液和密切夜间观察为主；若出现任何呼吸窘迫迹象，应立即升级就医。
