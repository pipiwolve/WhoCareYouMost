# 尿路感染危险信号

典型症状包括尿频、尿急、尿痛和下腹不适。需警惕发热寒战、腰背部疼痛、恶心呕吐等提示上尿路感染的表现。

## 需要尽快就医的信号

出现发热伴寒战、肋腰部疼痛、肉眼血尿、妊娠期泌尿症状、男性首次明显泌尿症状时，应尽快就医。

## 立即急诊

如出现高热不退、呕吐导致无法进水、意识改变、明显乏力或低血压表现，应立即急诊处理。

## 居家处理

症状轻且无危险信号时可增加饮水、避免憋尿并尽快门诊检查；不建议凭经验反复自行使用抗菌药。
