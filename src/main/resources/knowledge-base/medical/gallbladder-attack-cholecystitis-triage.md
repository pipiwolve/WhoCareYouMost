# 胆囊相关腹痛危险信号

胆囊急性炎症常见右上腹持续疼痛，可向右肩背放射，伴恶心、呕吐、发热，需与一般消化不良区分。

## 需要尽快就医的信号

右上腹突发持续性疼痛超过数小时，或伴发热、呕吐、黄疸时，应尽快就医，警惕急性胆囊炎或胆道感染。

## 立即急诊

若出现高热寒战、持续剧痛、意识改变、血压下降或无法进食饮水，应立即急诊。

## 居家处理边界

不建议在持续中重度腹痛情况下长期自行服止痛药观察，以免掩盖病情、延误手术或抗感染时机。
