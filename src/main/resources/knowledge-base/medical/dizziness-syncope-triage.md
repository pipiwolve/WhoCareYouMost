# 头晕与晕厥危险信号

头晕需要区分眩晕、黑蒙和站立不稳，晕厥需要关注发作前后是否有胸痛、心悸、抽搐、外伤或恢复延迟。

## 立即急诊

如果晕厥后长时间意识不清，或伴胸痛、呼吸困难、单侧肢体无力、言语含糊、持续呕吐、头部外伤，应立即急诊。

## 不建议仅居家观察的情形

首次不明原因晕厥、短期内反复晕厥、运动中或静息状态突发晕厥、老年人伴基础心脏病时，不建议仅线上观察。

## 居家处理

无危险信号且症状短暂缓解时，可先平卧休息、补液、缓慢起身并监测血压心率；若再次发作应及时就医。
