# 胸部感染危险信号

胸部感染可出现咳嗽、发热、胸痛、气促。需重点识别缺氧、呼吸衰竭或感染性全身恶化征象。

## 立即急诊

若出现呼吸困难明显、口唇发青发灰、意识改变、持续高热伴寒战、胸痛加重或血压不稳，应立即急诊。

## 需要尽快就医的情形

老年人出现新发意识差、活动耐量快速下降、咳嗽伴胸痛或咯血，应尽快线下评估肺炎或并发症风险。

## 居家观察边界

轻症可短期补液休息并监测体温呼吸状态；若二十四到四十八小时内无改善或逐步加重，应及时就医。
