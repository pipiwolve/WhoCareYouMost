# 心悸危险信号

心悸评估应关注持续时间、发作频率、心率快慢是否规则，以及是否伴胸痛、呼吸困难、头晕、近晕厥或血压异常。

## 立即急诊

如果心悸持续不缓解并伴胸痛、气促、明显头晕、晕厥、冷汗或面色苍白，应立即急诊排查严重心律失常或急性心脏事件。

## 需要尽快就医的信号

近一周内反复发作、夜间频繁发作、静息心率持续明显增快或减慢、既往有心脏病史者，应尽快心内科就诊。

## 居家处理

症状轻且短暂时可先停止运动、避免咖啡因和酒精、记录发作时间及心率；如发作加重或频率增加应尽快线下评估。
