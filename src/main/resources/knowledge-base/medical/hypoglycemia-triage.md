# 低血糖危险信号

低血糖可表现为心慌、出汗、手抖、饥饿、头晕、注意力下降，严重时可出现行为异常、抽搐或昏迷。

## 立即急诊

出现意识障碍、不能吞咽、抽搐、反复低血糖无法纠正，或低血糖发生于老年、妊娠、严重基础病患者时，应立即急诊。

## 居家紧急处理

患者清醒且能吞咽时，应尽快补充快速吸收糖分并在短时间内复测血糖；若症状未缓解或再次下降，应尽快就医。

## 后续复评

低血糖纠正后仍需复盘诱因，如漏餐、运动量变化、药物调整或饮酒，并尽快与医生确认是否需要调整治疗方案。
