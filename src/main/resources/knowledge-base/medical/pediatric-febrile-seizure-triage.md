# 热性惊厥危险信号

热性惊厥多见于发热儿童，发作时可出现肢体抽动、意识丧失或短时反应差。多数预后良好，但需规范识别高风险表现。

## 立即急诊

首次惊厥、发作持续超过五分钟、呼吸困难、单侧肢体抽动、二十四小时内反复发作或发作后超过一小时仍明显嗜睡时，应立即急诊。

## 发作现场处理

记录发作起止时间，保护头部并清理周围危险物；发作后置于侧卧位，勿强行按压肢体或向口中塞物。

## 后续评估

即使发作停止，也建议尽快儿科评估诱因和复发风险，避免仅凭一次退热后自行长期观察。
