# 头痛危险信号

头痛需要区分原发性头痛和继发性危险头痛，重点关注是否为突发最严重头痛，以及是否伴发热、颈部僵硬、神经功能异常或意识改变。

## 立即急诊

如果出现爆发样剧烈头痛、头痛伴肢体无力或言语不清、抽搐、持续呕吐、外伤后头痛加重、意识模糊，应立即急诊排查脑出血、卒中等紧急情况。

## 需要尽快就医的信号

头痛持续超过七十二小时仍无缓解、近期频率明显增加、夜间痛醒、伴高热或视力变化时，应尽快神经内科或急诊评估。

## 居家处理

既往类似发作且无危险信号时，可短期休息、补液、避免强光和噪音；若止痛后仍反复发作，应线下明确病因。
