# 阿片过量危险信号

阿片类药物过量可导致呼吸抑制和意识障碍，是可迅速致命的紧急状态。

## 立即急诊

出现针尖样瞳孔、叫不醒或意识丧失、呼吸浅慢或暂停、喉部咕噜声、皮肤苍白青冷、肢体松软时，应立即呼叫急救。

## 现场处理

若具备条件可立即使用纳洛酮并持续观察，尽量保持气道通畅，将患者置于侧卧位以降低误吸风险。

## 不建议等待观察

无法区分“嗜睡”与“过量”时应按过量处理，优先急救，不应单纯等待其自行清醒。
