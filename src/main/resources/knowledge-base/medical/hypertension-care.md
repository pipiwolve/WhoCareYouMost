# 高血压居家监测要点

血压评估应结合连续多次测量和症状判断，不应仅凭单次读数决定处理。测量前应安静休息，避免刚运动或情绪激动后立即测量。

## 立即急诊

若血压明显升高并伴剧烈头痛、胸痛、呼吸困难、神经功能异常、视力急剧下降或意识改变，应立即急诊。

## 需要尽快门诊调整的情形

近期晨起血压持续偏高、波动明显增大、现用药后仍长期控制不佳，应尽快复诊评估依从性、生活方式和用药方案。

## 居家处理

规律监测血压、限制高盐饮食、控制体重、避免熬夜和大量饮酒；未经医生评估不建议频繁自行加减降压药。
