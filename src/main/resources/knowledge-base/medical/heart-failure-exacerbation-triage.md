# 心衰加重识别要点

心衰加重常表现为活动后或静息气促、乏力、下肢水肿、夜间憋醒或平卧困难，需要尽早识别急性失代偿。

## 立即急诊

若出现突发或非常严重的呼吸困难、不能平卧、胸痛、意识改变、血压不稳或严重乏力，应立即急诊。

## 需要尽快复诊的情形

近期气促加重、下肢水肿上升、活动耐量明显下降、夜间症状增加时，应尽快心内科复评。

## 居家管理边界

有明确心衰病史者可按医嘱监测症状变化，但出现进行性恶化不应自行延迟就医。
