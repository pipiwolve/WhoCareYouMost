# 胸痛危险信号

胸痛伴呼吸困难、出汗、恶心、呕吐、濒死感、向左肩或下颌放射痛时，应高度警惕急性冠脉综合征等紧急情况。

## 立即急诊

如果胸痛持续超过十五到二十分钟，或者伴随大汗、恶心、明显乏力、呼吸困难，应立即前往急诊或呼叫急救，不应自行观察过久。

## 不建议仅居家观察的情形

出现持续性胸痛、活动后加重、静息时也不缓解、伴冷汗或心悸时，不建议只依赖线上建议或家庭常备药，应尽快线下评估。
